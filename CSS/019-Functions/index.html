CSS Functions

attr( ) : Uses the attributes defined on HTML element. It's an example 

                  a:hover::after {
                    content: ' (source: ' attr(href) ') ';
                    color: #c7c7ff;
                  }

calc( ) : The math function that performs basic arithmetic calculations on numerical values. There are some uses of this function: 

                  width: calc(100% - 30px);
                  width: calc(100% / 4);
                  font-size: calc(1.5rem + 3vw);

max( ) : Uses the largest value, from a comma-separated list of values. There are some good uses: 

                  width: max(20vw, 100px);
                  height: max(30vh, 200px);
                  font-size: max(1.5rem, 3vh);

min( ) : Uses the smallest value, from a comma-separated list of values. There are some good uses: 

                  width: min(200px, 50vw);
                  height: min(300px, 40vh);
                  font-size: min(2rem, 3vh);

clamp( ) : Calculates the central of a minimum, central, and maximum values. There are some examples: 

                  width: clamp(20rem, 30vw, 70rem);
                  height: clamp(10vh, 20em, 100vh);
                  font-size: clamp(1.5rem, 3vh, 2rem);

var( ) : Inserts the value of a custom property. Here an example; 

                  :root {
                    --bg-color: purple;
                    --txt-color: blue;  
                    --main-padding: 20px;  
                  }
                  
                  .div {
                    background-color: var(--bg-color);
                    color: var(--txt-color);
                    padding: var(--main-padding);
                  }

url( ) : Uses a file from the specified URL. Some examples: 

                  body{
                    background-image: url(img/background2.jpg);
                  }

                  li::before {
                    content: url(tick.gif);
                  }

rgb( ) : Defines colors according to red-green-blue model and transparency. 

                  color: rgb(109, 109, 209, 80%);
                  background-color: rgb(199, 199, 255);

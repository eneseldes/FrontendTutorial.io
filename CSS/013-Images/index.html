Adding Images
Back-ground image: url();
The code used to add a photo inside a field is as follows: 
                    .imgArea { 
                        background-image: url( images/example.jpg );
                    }
The class name of the div containing the image is "imgArea". You should add this code inside the css file. However, we realize that the picture is repeating and does not look good aesthetically. Let's change other properties step by step.


Properties
Background-repeat, background-size, background-position, background-attachment
1 - The code for background-repeat feature is follows: 
                    .imgArea { 
                        background-image: url( images/example.jpg );               
                        background-repeat: no-repeat;
                    }
"no-repeat" allows the image to be displayed only once. "repeat-x" makes it repeat horizontally and "repeat-y" makes it repeat vertically.


2 - The code for background-position feature is follows: 
                    .imgArea { 
                        background-image: url( images/example.jpg );
                        background-repeat: no-repeat;
                        background-position: center;
                    }

center: Centers the background image both horizontally and vertically within the element's background area.



left: Positions the background image at the left edge of the element's background area.

right: Positions the background image at the right edge of the element's background area.

top: Positions the background image at the top edge of the element's background area.

bottom: Positions the background image at the bottom edge of the element's background area.



These values determine where the background image is placed within the element's background area without altering the other alignment.
3 - The code for background-size feature is follows: 
                    .imgArea { 
                        background-image: url(images/example.jpg);
                        background-repeat: no-repeat;
                        background-position: center;
                        background-size: cover;
                    }

cover: scales the image to fully cover the background area, potentially cropping parts of the image.



contain: scales the image to fit within the background area without cropping, ensuring the entire image is visible.



4 - The code for background-attachment feature is follows: 
                    .imgArea { 
                        background-image: url(images/example.jpg);
                        background-repeat: no-repeat;
                        background-position: center;
                        background-size: cover;
                        background-attachment: scroll;
                    }
It is difficult to show this feature in a narrow area through the picture, let's explain it in terms of concepts:


scroll: scrolls the background image along with the element's content.

local: is used less frequently and makes the background image scroll with individual elements that have their own scrolling.

fixed: keeps the background image fixed relative to the viewport, creating a consistent background effect as the content scrolls over it.

Positioning
(static, relative, absolute, fixed, sticky)
static
Default value of every element. Elements which is static, cannot be affected by top, left, right or bottom properties.
relative

An element with 'relative' value is positioned relative to its exact position. 

                  .relative-box:hover{
                    background-color: var(--secondaryBoxColor);
                    position: relative;
                    top: 50px;
                    left: 50px;
                  }
Hover The Box Below
Relative Box
Footnote

Setting position with relative, adjusts the element away from its exact position. In this example, box is 50px away from its exact position.

absolute

An element with 'absolute' value is positioned relative to the body element and also do not take place on the web page. 

                  .absolute-box{
                    position: absolute;
                    right: 50px;
                  }
                  .example-absolute:hover{
                    .absolute-box{
                        background-color: var(--secondaryBoxColor);
                    }
                    position: relative;
                  }
Absolute Box
Footnote

The dashed box above has a default position value(static) before you hover it. In this situation, absolute box adjust the position relative to the body.

However, when you hover the dashed box, it sets its position value as 'relative'. If you want to position an element relative to its ancestor, that element's ancestor must be set as 'relative'.

sticky

An element with 'sticky' value "sticks" to screen as you coded to be, and stays at the viewport when website is scrolled. 

                  .sticky-box{
                    background-color: var(--primaryBoxColor);
                    position: sticky;
                    top: 0px;
                  }
Sticky box
Footnote

If you see this text, you have already understood what sticky box does. You are going to see, when you keep scrolling, it is been sticky to top:0px. It keeps its position until the end of its ancestor.

fixed

An element with 'fixed' value always stays in the same place even though the page is scrolled. It is always on user's viewport.

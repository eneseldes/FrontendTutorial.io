Grid Layout
grid-container

Using the Grid layout begins with setting a grid container. Grid container includes:

display: grid : generates a block-level grid

grid-template-columns : defines how wide columns should be.

grid-template-rows : defines how long columns should be.

grid-auto-columns : specifies the size of any auto-generated grid column tracks.

grid-auto-rows : specifies the size of any auto-generated grid row tracks.

grid-column-gap : puts some space between the columns.

grid-row-gap : puts some space between the rows.

gap : puts space between both the columns and rows.

fr : the fractional unit fr is using instead of percentage.

Let's put them together;

                  .grid-container{
                    display: grid;
                    grid-template-columns: repeat(3, 1fr);
                    grid-template-rows: 3fr 5fr;
                    gap: 10px;
                    background-color: #7364d273;
                  }
                  .grid-item{
                    background-color: #858ae388;
                  }

Now we have 3 columns each width is equal, 2 rows split in proportions of 3:5 and 10px space between columns and rows. ( I also added width, height and 10px padding in container.) Simple grid layout. Nothing complex yet !

grid-items

Grid item is the children of the grid container.

grid-column : Shorthand for grid-column-start + grid-column-end. Specifies a grid item's size and location.

grid-row : Shorthand for grid-row-start + grid-row-end. Specifies a grid item's size and location.

grid-area : This property can be used as shorthand for row-start + column-start + row-end + column-end + column-end. 

                  .grid-container{
                    width: 55%;
                    min-height: 200px;
                    display: grid;
                    background-color: #7364d273;
                    grid-auto-rows: minmax(100px,auto);
                    grid-template-columns: repeat(3, 1fr);
                    gap: 10px;
                    padding: 10px;
                  }
              
                  .grid-container div{
                    text-aligns: center;
                    border: 1px solid black;
                    background-color: #858ae3b6;
                  }
              
                  .grid-item-3{
                    grid-column: 3/4;
                    grid-row: 1/3;
                  }

                  .grid-item-4{
                    grid-column: 1/3;
                    grid-row: 2/3;;
                  }

                  .grid-item-5{
                    grid-area: 3/1/4/4;
                  }

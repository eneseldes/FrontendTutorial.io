Pseudo classes

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element.

:link - Matches links that have not yet been visited.

:visited - Matches links that have been visited.

:any-link - Matches an element if the element would match either :link or :visited.

:hover - Matches when the user hovers over an element with the pointer.

:active - Matches when an item is being activated by the user. For example, when the item is clicked on.

:focus - Matches when an element has focus. For example, when the user clicks or taps on an element or selects it with the keyboard's Tab key.

:first-child - Select the specified selector, only if it is the first child of its parent.

:nth-child(n) - Matches every element that is the nth child of its parent. n can be a number, or a keyword (odd or even).

Here an example:

                  HTML
                  <a href="https://www.aybubiltek.com/" target="_blank">This is a link</a>

                    </p>
                    <p> This is an example paragraph 2. Lorem ipsum dolor sit, amet consectetur adipisicing elit.
                    </p>
                    <p> This is an example paragraph 3. Lorem ipsum dolor sit, amet consectetur adipisicing elit.
                    </p>
                  </div>
                  CSS
                  a:link{
                    color: purple;
                  }
                  a:visited{
                    color: blue;
                  }
                  a:hover{
                    color: green;
                  }

                    color: orange;
                  }

Pseudo elements

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element.

::after - Creates a pseudo-element that is the last child of the selected element.

::before - Creates a pseudo-element that is the first child of the selected element.

::first-letter - Applies styles to the first letter of the first line of a block-level element.

::first-line - Applies styles to the first line of a block-level element.

Here an example:

                  HTML
                  <div class="pseudo2">
                  <p>This is an example paragraph. Lorem ipsum dolor sit amet consectetur, adipisicing elit. Totam voluptate laboriosam, inventore culpa possimus voluptatibus, ab aut veritatis ea cupiditate nesciunt, animi blanditiis esse est accusantium nemo molestiae magni nihil.
                  </p>
                  <a href="https://www.aybubiltek.com/" target="_blank">This is a link</a>
                  </div>
                  CSS
                  .pseudo2 a::before {             
                    content: "Click here: ";
                    color: #cec2ff;
                  } 
                  .pseudo2 a::after {               
                    content: ' (' attr(href) ') ';
                    color: #beffca;
                  }
                  .pseudo2 a:any-link{            
                    color: #a8deff;
                  }
                  .pseudo2 p{
                    color: rgb(244, 242, 255);
                  }
                  .pseudo2 p::first-letter{
                    font-size: 2em;
                  }
                  .pseudo2 p::first-line{
                    font-weight: bold;
                    text-decoration: underline;
                  }

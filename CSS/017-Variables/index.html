Variables and var()
What Is Variables?

Variables is used to hold a particular value and use that value wherever we want.

For example, the box class' background color is declared by a variable in the code below: 

                    .box{
                        background-color: var(--bgColor);
                    }

In CSS, a double dash is used to declare variables such as --main-color:blue. CSS variable names are case sensitive. The --main-color and --Main-Color variables are perceived by browsers as different variable names.

To use var() in CSS, a variable must be defined. You can specify global and local variables.

Global Variables

A global variable can be called from any line of of any code block. Global variables must be declared within the :root psuedo class. 

                    :root{
                        --mycolor: purple;
                        --fontWeight: 900;
                        --borderRadiusValue: 10px;
                    }
                    .box1{
                        background-color: var(--mycolor);
                        font-weight: var(--fontWeight);
                        border-radius: var(--borderRadiusValue);
                    }
This is a box
Local Variables

Contrary to the global variables, local variables can be called just in the code block it declared. Furthermore, local variables have the ability to override global variables. (See the example below)

These variables are defined within an element, not in :root{}. 


                    :root{
                        --mycolor: purple;
                        --fontWeight: 900;
                        --borderRadiusValue: 10px;
                    }
                    .box2{
                        --mycolor: #7364d273;
                        --borderRadiusValue: 10px;
                        background-color: var(--mycolor);
                        border-radius: var(--borderRadiusValue);
                    }
This is a box
Important Footnote

Even though '--mycolor' variable equals to purple in the :root pseudo class, we override the purple color and set the box's background color to color #7364d273 because of the local variable. And also we added a border radius not with a certain value but with a variable.

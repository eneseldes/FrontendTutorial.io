Units & Sizes
Absolute And Relative Length Units
Many CSS properties like width, margin, padding, and font-size take a length, and CSS has many different ways to express length. There are two general kinds of units used for length and size in CSS: absolute and relative.

px : Pixels are one of the most common length units in CSS. The px unit is absolute. 1px is equal to a single point (“dot”). For example: paragraph will take up 400px pixels in width and font size will be 24px.

                  p{
                    font-size: 24px;
                    width: 400px;
                 }

em : The em unit is relative. This unit when used with the font-size property, em inherits the font-size from its parent element.

Example: p contents will be (20*0.5) 10px.

                  .container {
                    font-size: 20px;
                  }

                  .container p {
                    font-size: 0.5em;
                  }

rem : The rem unit is relative to the font size of the root element. For websites, the root of the document is the html element. (e.g <html> or :root {} ). In most browsers, the default font size is 16px. In this case, p is (16*2) 32px. 

                  p {
                    font-size: 2rem;
                  }

% : Percentages relative to the parent's size. Such as

                  .container {
                    width: 400px;
                  }
                  
                  .container p {
                    width: 75%;
                  }

vh and vw : View height is 1% of the height of the viewport. View width is 1% of the width of the viewport. 

                  body {
                    width: 100vw;
                    height: 100vh;
                  }

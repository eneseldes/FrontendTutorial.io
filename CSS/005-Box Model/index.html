Box Model
Understanding The Spaces One By One
You can reach the screen below by pressing CTRL+SHIFT+I.

Content(Blue Area): Our exact element. image, text, heading etc.

Padding(Green Area): Space around our element, but, inside of element's borders.

Border: The line that circle around the element.

Margin(Orange Area): space around our element, but, outside of element's borders.

box-sizing Property

box-sizing is a property that inform the website of how to calculate our elements' side lengths and have to values:

border-box value allows website to consider the padding and border when the website calculates the length of an element.

However, with content-box, only content's itself are being calculated.


To make it clear, here is an example: 
                    .content-box{
                      box-sizing: content-box;
                      margin: 5px auto;
                      padding: 50px; 
                      border-radius: 40px; 
                      border: 1px white solid;
                      width: 250px;
                    }
                    .border-box{
                      box-sizing: border-box;
                      margin: 5px auto;
                      padding: 50px;
                      border-radius: 40px; 
                      border: 1px white solid;
                      width: 250px;
                    }

This is a box with 'content-box' sizing.


This is a box with 'border-box' sizing.


As you can see in the box above, both boxes have 250px width. However, content-box is much more bigger because it calculates just the content as 250px, and there are also padding and border outside of it. To create easy to organise web pages, box-sizing is more effective.

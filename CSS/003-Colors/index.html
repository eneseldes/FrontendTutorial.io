CSS Colors

The color data type represents a color. A color may also include a transparency value. With CSS, colors can be specified in different ways, such as RGB/ HEX/ HSL/ HWB values or just by color names.

RGB (red, green, blue); each parameter defines the strength of the color as a number between 0 and 255. 

                  color: rgb(225, 0, 0);
                  color: rgb(0, 225, 0);
                  color: rgb(0, 0, 225);
                  color: rgb(225, 225, 225);
                 color: rgb(0, 0, 0);

HXL A hexadecimal color is specified with: #RRGGBB. [RR (red), GG (green) and BB (blue)]. 

                   color: #FF0000;
                   color: #00FF00;
                   color: #0000FF;
                   color: #FFF;
                  color: #000;

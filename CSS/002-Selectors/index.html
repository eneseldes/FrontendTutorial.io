Selectors
Universal Selector: *

This selector affects every element on the html. That is because it is called 'universal'. 

                  * {
                    margin: 0;
                    padding: 0;
                    box-sizing: border-box;
                  }
Footnote

The code above is a general thing that must be done. The default value of these properties causes collapse after we set our own value. So, at the beginning of every project, the code above must be declared.

General Selectors

Elements can be selected with their tag names. However, this method will affect all the elements in question. 

                    body{
                      font-family: Arial, sans-serif;
                    }
                    p{
                      color: black;
                    }
                    div{
                      width: 400px;
                      height: 500px;
                    }

According to the code above, all the text's font family will be set as arial in the body element. All the p element's text color will be black. Also, every div in the web page will have 400px width and 500px height. This kind of selectors do not select the elements specifially.

Class and ID Selector

In CSS, classes are being selected with a dot in the beginning of the class name.

Similarly, # is being used to select IDs. 

                  HTML
                  <div class="box classBox">"classBox" div</div>
                  CSS
                  .classBox{
                    width: 150px;
                    height: 150px;
                    background-color: #003554;
                  }
"classBox" div


Wrong Way Of ID Usage 
                  HTML
                  <div id="box idBox">Div with multiple IDs</div>
                  CSS
                  #idBox{
                    width: 80%;
                    height: 150px;
                    background-color: #003554;
                  }
This is the div styled in the code above. As you can see nothing changed.


Proper Way Of ID Usage 
                  HTML
                  <div id="idBox">This is a div affected by a properly used ID.</div>
                  CSS
                  #idBox{
                    width: 80%;
                    height: 150px;
                    background-color: #003554;
                  }
This is a div affected by a properly used ID.
Footnote
Be careful while using ID. It MUST be unique. If an element is being forced to have multiple IDs, none of the IDs are going to implement as seen in the examples above.
Descendant Selector

Selecting process may be done more specifically.

                  HTML
                  <div class="box ancestorDiv">
                    <p>This is a p element in the ancestor div.</p>
                    <div class="box childDiv">
                      <p>This is a p element in the child div.</p>
                    </div>
                  </div>
                  CSS
                  .ancestorDiv{
                    margin: 0 auto;
                    background-color: #7364d273;
                    width: 100%;
                    height: 150px;
                    text-align: center;
                  }
                  .childDiv{
                    margin: 10px auto;
                    background-color: #858ae3b6;
                    width: 70%;
                    height: 75px;
                    text-align: center;
                  }
                  .ancestorDiv p{
                    color:darkgoldenrod;
                  }
                  .ancestorDiv .childDiv p{
                    color:floralwhite
                  }

This is a p element in the ancestor div.

This is a p element in the child div.

Footnote

Pay attention to the boxes and the codes above. Normally, all the paragraphs in the ancestor div must be darkgoldenrod. However, the paragraph inside of the child div is floralwhite while the other paragraph is darkgoldenrod. This is because the second row of the code is a more specific selector. As a result, it overwrote the first code.

Attribute Selector

Attribues can also be selected.

                  HTML
                  <a target="_blank" href="https://www.aybubiltek.com/">
                  Link with _blank</a>
                  <br />
                  <a href="https://www.aybubiltek.com/">Normal Link</a>
                  CSS
                  a[target="_blank"] {
                    color: #7364d273;
                  }
Link with _blank
Normal Link

Footnote

First link has a _blank attribute but second one does not. Selector affected the link which has _blank attribute.

Pseudo -Classes and Pseudo-Elements

These structures are also declared in the CSS file with a selector method. 

                  HTML
                  <div class="box pseudoBox">
                    Lorem ipsum dolor sit amet consectetur adipisicing elit. Quia
                    eligendi necessitatibus officiis quam excepturi est!
                  </div>
                  CSS
                  .pseudoBox{
                    background-color: var(--primaryBoxColor);
                    margin: 0 auto;
                    text-align: center;
                    width: 250px;
                    height: 80px;
                    padding: 12px;
                  }
                  .pseudoBox:hover{
                    background-color: var(--secondaryBoxColor);
                  }
                  .pseudoBox::first-line{
                    color: sandybrown;
                  }
Hover this box to see what is changing! Awesome, right!

Specificity

Every selector has specificity. The selector which has greater value of specificity overwrites the selector that has lower specificity.

Here are the specificity values of the structures in order:


Inline Styles: 1000

IDs: 100

Classes, Pseudo-Classes,Attribute Selectors: 10

Elements, Pseudo-Elements: 1

Footnote

Inline styles means the style which is done in the element tag in HTML.

To calculate specificity: Start at 0, add the values above when you see in the selector structure. If they are in a group, add all the values of the structures(inline styles, classes, etc.).

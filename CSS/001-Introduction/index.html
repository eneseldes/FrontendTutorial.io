What Does CSS Do?

CSS is the part of the Frontend which we add style to the elements, and also organize our web page.

First thing must be done is link the HTML to the CSS. This process is being writen inside of the head tag in HTML. 

                    HTML
                      <head>
                        ...
                        ...
                      <link rel="stylesheet" href="main.css">
                      </head>

To affect the elements with the codes in CSS, elements must be identified with "class" or "ID"-will be explained- to call the elements to the CSS. 


                    HTML
                      <div class="classBox">This is a div which has class name "classBox".</div>
                    CSS
                      .classBox{
                        margin: 0 auto;
                        padding: 20px;
                        width: 80%;
                        height: 150px;
                        background-color: var(--primaryBoxColor);
                        border-radius: 8px;
                        display: grid;
                        place-content: center;
                        text-align: center;
                        }
This is the div which has class name "classBox".

Another example is:
                    HTML
                      <p id="p1">
                        Width and height of the elements, <span class="bgColor">background colors</span>, <span class="borderRadius">border radius'</span>, <span class="item">styling the texts</span> <span class="etc">etc.</span> are being done in the CSS.
                      </p>
                    CSS
                      #p1{
                        font-size: 2rem;
                      }
                      .bgColor{
                        background-color: var(--primaryBoxColor);
                      }
                      .borderRadius{
                        background-color: var(--secondaryBoxColor);
                        border-radius: 8px;
                      }
                      .etc{
                        font-weight: 600;
                        text-shadow: 1px 1px 1px whitesmoke;
                        font-size: xx-large;
                      }

Width and height of the elements, background colors, border radius', styling the texts etc. are being done in the CSS.

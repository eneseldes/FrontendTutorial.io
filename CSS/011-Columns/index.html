Columns And Its Properties
column-count, column-width

column-count: Aligns the elements side by side.

column-width: Sets the width of every column. 

                    .columns1{
                        column-count: 3;
                    }
                    .columns2{
                        column-count: 3;
                        column-width: 300px;

                        /*This properties also can be used like the code below*/

                        columns: 3 200px;
                    }

                        columns: 4 250px;
                        column-rule: 2px solid var(--primaryBoxColor);

Deleting The First Margin

If you pay attention to the first row of the example, you can see it is empty. This is because of the first paragraph's margin. To prevent this from happening, we must delete upper margin. 

                    .deleting-margin{
                        columns: 3 200px;
                    }
                    .deleting-margin p{
                        margin-top: 1rem;
                    }

break-inside

If you change the size of the web page, you can see the space belong to 'topic' is seperated and involved to another column. break-inside property must be used on that 'involved' element with 'avoid' value to prevent this situation. 

                    .column4{
                        columns: 4 250px;
                        break-inside: avoid;
                    }
                    h4{
                        background-color: tomato;
                        font-size: larger;
                        height: 100px;
                        break-inside: avoid;
                    }

column-span, white-space

column-span: Creates a text which seperate the columns like a heading.

white-space: 'nowrap' value of this property allows us to make elements whole. As a result, our element cannot be break into row below. 

                    .heading-h{
                        color: tomato;
                        column-span: all;
                        text-align: center;
                        background-color: #211e2d;
                    }
                    .spanned{
                        white-space: nowrap;
                    }

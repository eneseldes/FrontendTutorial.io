HTML Lists

HTML lists allow web developers to group a set of related items in lists.

Unordered HTML List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default: 

                  <ul>  
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                  </ul>

Output:

  • Item 1
  • Item 2
  • Item 3
    • Ordered HTML List

    An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

    The list items will be marked with numbers by default:

                      <ol>     
                    <li>First item</li>
                    <li>Second item</li>
                    <li>Third item</li>
                  </ol>

    Output:

    1. First item
    2. Second item
    3. Third item

    You can change the list sorting type or start from any sign you want. 

                      <ol type="a" start="3">
                    <li>First item</li>
                    <li>Second item</li>
                    <li>Third item</li>
                  </ol>

    This code starts from c:

    Output:

    1. First item
    2. Second item
    3. Third item
    Definition List

    The <dl> tag is used in conjunction with <dt> (defines terms/names) and <dd> (describes each term/name). 

                      <h2>Definition List (dl):</h2>
                  <p>A definition list displays elements in definition form like in dictionary.</p>
                  <dl>    
                      <dt>HTML</dt>
                      <dd>HyperText Markup Language</dd>
                      <dt>CSS</dt>
                      <dd>Cascading Style Sheets</dd>
                      <dt>JS</dt>
                      <dd>JavaScript</dd>
                  </dl>

    Output:

    Definition List (dl):

    A definition list displays elements in definition form like in dictionary.

    Nested Lists

    You can also nest lists inside other lists for hierarchical structures. 

                      <ul>
                      <li>Main Item 1
                        <ul>
                          <li>Sub Item 1</li>
                          <li>Sub Item 2</li>
                        </ul>
                      </li>
                      <li>Main Item 2</li>
                      <li>Main Item 3
                        <ul>
                          <li>Sub Item 3</li>
                          <li>Sub Item 4</li>
                        </ul>
                      </li>
                  </ul>

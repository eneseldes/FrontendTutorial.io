Adding Media
Adding Image

To add an image, use the <img> tag and provide the src attribute with the path to the image file.

alt attribute is an alternative description for the image not showing.

The image size is adjusted according to the width and height attributes. 

                 <img src="images/aybu.jpg" alt="A logo of AYBU" height="200" width="500" />

Adding Video

The HTML <video> element is used to show a video on a web page.

Video Attributes

src: Specifies the URL or file path of the video file.

controls: Adds playback controls (play, pause, volume, etc.) to the video.

autoplay: The video starts playing automatically when the page loads.

loop: The video automatically restarts when it reaches the end.

muted: The video is muted by default. 

                  <video src="video/aybu_tanitim.mp4" controls height="250" width="500" muted>
                  <!-- If the video doesn't play or is not supported, you can provide an alternative text.-->
                  Your browser does not support the video tag.
                  </video >

Adding Audio

To add an audio file, use the <audio> tag and provide the src attribute with the path to the audio file.

The 'controls' attribute will add playback controls like play, pause, volume, etc.

Similar commands apply when adding sounds, autoplay, muted, loop etc. 

                  <audio src="path/to/audio.mp3" controls>
                  <!-- If the audio doesn't play or is not supported, you can provide an alternative text. -->                    
                  Your browser does not support the audio element.
                  </audio >

Form Input Attributes
HTML Input Tag

The HTML <input> tag defines the field where the user can enter data. The type attribute determines what type of user input is taken. 

                    <input type="text" name="firstname">

Output:

  • type - determines the type of input the <input> tag takes
  • name - specifies the name of the input which is what the data is named as when submitting the data to a server.
    • Different Input Types

    The various types of input tags available in HTML5 are:

    1. text - creates a single-line text fields (default)
    2. button - creates a button with no default functionality
    3. checkbox - creates a checkbox
    4. color - creates a color picker
    5. date - creates a date picker
    6. datetime-local - creates a date and time picker
    7. email - creates an input field that allows the user to input a valid email address
    8. file - creates an input field that lets the user upload a file or multiple files
    9. image - creates a button using an image
    10. password - creates an input field that lets the user enter information securely
    11. radio - creates a radio button
    12. range - creates a range picker from which the user can select the value
    13. reset - creates the button which clears all the form values to their default value
    14. submit - allows user to submit form to the server
    15. tel - defines the field to enter a telephone number
    16. time - creates an input field that accepts time value
    17. url - lets the user enter and edit a URL

    Some Other Attributes
  • value: specifies the intial value for the control(type = "checkbox" or type = "radio")
  • placeholder: specifies a short hint that describes the expected value
  • required: specifies that an input field must be filled out before submitting the form
  • multiple: allows users to choose more than one value from selection
  • maxlength: defines the maximum number of characters allowed in a text field
  • accept: specifies a comma-separated list of content types that the server accepts
  • autocomplete: specifies for enabling or disabling of autocomplete in <input > element
  • checked: if type = "radio" or type = "checkbox" it will already be selected when the page loads
  • readonly: sets the input control to read-only. It won't allow the user to change the value
  • disabled: disables the input control. The button won't accept changes from the user
  • pattern: specifies a regular expression to check the input value against


    • text

    The input type text is used to create single-line text fields. It is the default input type. 

                       <label for="name">Search: </label>
                   <input type="text" id="name">

    Output:

    button

    The input type button is used to create a button with no default functionality. 

                       <input type="button" value="Click Me!">

    Output:

    checkbox

    The input type checkbox is used to create a checkbox input.

                      <input type="checkbox" id="subscribe" value="subscribe">
                  <label for="subscribe">Subscribe to channel!</label><br>

    Output:


    color

    The input type color is used to create an input field that lets the user pick a color. 

                        <input type="color" id="background">
                    <label for="background">Background Color</label>

    Output:

    date

    The input type date is used to create an input field that lets the user input a date using the date picker interface from the browser. 

                      <label for="birthday">Birthday:</label>
                  <input type="date" id="birthday">

    Output:

    datetime-local

    The input type datetime-local is an HTML input type used for date and time selection on web pages. It allows users to pick a specific date and time value. 

                      <label for="meeting-time">Choose a time for your appointment:</label>
                  <input type="datetime-local" id="meeting-time" >

    Output:

    email

    The input type email is used to create an input field that allows the user to input a valid email address. 

                      <label for="email">Enter your email:</label>
                  <input type="email" id="email">

    Output:

    This input field throws an error if the value provided is not a valid email.

    file

    The input type file is used to create an input field that lets the user upload a file or multiple files from their device. 

                      <label for="file">Choose the file/files:</label>
                  <input type="file" name="file" id="file">

    Output:

    The accept attribute provides file limitation by specifying the type of file that can be added. The following code only accepts pdf files: 

                      <label for="file">Choose the file/files:</label>
                  <input type="file" name="file" id="file" accept=".pdf">

    Output:

    image

    The input type image is used to create a button using an image. 

                        <input type="image" src="/submit.png" alt="submit">

    Output:

    password

    The input type password is used to create an input field that lets the user enter information securely. 

                      <label for="password">Password:</label>
                  <input type="password" id="password">

    Output:

    radio

    The input type radio is used to define a radio button. Radio buttons are defined in a group. Radio buttons let users pick one option out of a group of options. 

                        <form>
                      <input type="radio" id="male" name="gender" value="male">
                      <label for="male">male</label>
                      <input type="radio" id="female" name="gender" value="female">
                      <label for="female">female</label>
                    </form>

    Output:

    range

    The input type range is used to create a range picker from which the user can select the value. User can select a value from the range given. It has a default range from 0 to 100. 

                        <label for="range">Select value: </label>
                    <input type="range" id="range" value="90">

    Output:

    reset

    The input type reset defines the button which clears all the form values to their default value. 

                        <form>
                      <label for="name">Name:</label>
                      <input id="name" type="text" /><br />
                      <input type="reset" value="Reset" />
                    </form>

    Output:


    submit

    The input type submit is used when the user submits the form to the server. It is rendered as a button. 

                        <input type="submit" value="submit">

    Output:

    tel

    The input type tel is used to define the field to enter a telephone number.

    The telephone number is not automatically validated to a particular format as telephone formats differ across the world. It has an attribute named pattern used to validate the entered value. 

                      <label for="phone">Phone Number:</label>
                  <input
                    type="tel"
                    id="phone"
                    placeholder="0555-555-5555"
                    pattern="[0-9]{4}-[0-9]{3}-[0-9]{4}"/>

    Output:

    time

    The input type time attribute creates an input field that accepts time value.

    It allows users to add time in hours, minutes, and seconds easily. 

                        <label for="time">Select Time:</label>
                    <input type="time" id="time" />

    Output:

    url

    The input type url is used to let the user enter and edit a URL. 

                        <label for="url">URL:</label>
                    <input
                      type="url"
                      id="url"
                      placeholder="https://example.com"
                      pattern="https://.*" />

    Output:

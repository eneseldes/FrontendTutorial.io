Heading Tags

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading. 

                    <h1> Heading 1 <h1>
                    <h2> Heading 2 <h2>
                    <h3> Heading 3 <h3>
                    <h4> Heading 4 <h4>
                    <h5> Heading 5 <h5>
                    <h6> Heading 6 <h6>

When you write it in your own compiler, it looks exactly like this:


Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Note:

  • Browsers automatically add some white space (a margin) before and after a heading.

  • Use HTML headings for headings only. Don't use headings to make text BIG or bold. Because Headings clearly express the structure of the document, increasing the understandability of the document.

    HTML Quotation and Citation Elements


    <abbr> Defines an abbreviation or acronym

    <address> Defines contact information for the author/owner of a document

    <bdo> Defines the text direction

    <blockquote> Defines a section that is quoted from another source

    <cite> Defines the title of a work

    <q> Defines a short inline quotation

    Usage of <hr> and <br> Tags


    The <hr> tag stands for "horizontal rule" and is used to create a horizontal line in a document. It is often used to visually separate different sections of content or to create a visual break between elements. The above line was created with the <hr> tag.

    The <br> tag stands for "line break" and is used to create a new line within a block of text. It is used to force content to start on a new line without creating a new paragraph.

HTML Table

The HTML table tag (<table>) is used to represent data in a structured way by creating a table. 

                    <table border="1">
                      <tr>
                        <th>Name</th>
                        <th>Age</th>  
                        <th>Country</th>
                      </tr>
                      <tr>
                        <td>Harry Depp</td>
                        <td>28</td>
                        <td>Britain</td>
                      </tr>
                      <tr>
                        <td>John Smith</td>
                        <td>35</td>
                        <td>USA</td>
                      </tr>
                      <tr>
                        <td>Ram Krishna</td>
                        <td>19</td>
                        <td>Nepal</td>
                      </tr>
                    </table>

Output:

Name Age Country
Harry Depp 28 Britain
John Smith 35 USA
Ram Krishna 19 Nepal

In the above example, you can see we have used multiple tags to create a table in HTML.

The border = 1 property is given to define the boundaries of the table. (Table borders will be explained at the end of this lesson. You can review to border section by clicking here.)



Tags

  1. <table> - Defines a table
  2. <tr> - Defines a row in a table
  3. <td> - Defines a cell in a table
  4. <th> - Defines a header cell in a table
Table Tag <table>

The <table> tag is used to define a table.

It consists of table cells inside rows and columns.

                    <table>
                      ...
                      ...
                    </table>
Table Row <tr>

The <tr>tag is used to define a row in a table.

                    <table>
                      <tr>
                      ...
                      </tr>
                    </table>

The table row can include either table heading <th> or table data <td>. 

                    <table border="1">
                      <tr>
                        <th>Name</th>
                        <th>Country</th>
                      </tr>
                      <tr>
                        <td>Prasanna</td>
                        <td>Nepal</td>
                      </tr>
                      <tr>
                        <td>Simon</td>
                        <td>USA</td>
                      </tr>
                    </table>
Name Country
Prasanna Nepal
Simon USA
Table Cell <td>

The <td> tag is used to define table cells (data). The table cells store data to be displayed in the table.

Table cells are generally inside the table row or table headers. 

                  <table border="1">
                  <tr>
                    <td>Apple</td>
                    <td>Mango</td>
                    <td>Orange</td>
                  </tr>
                  </table>
Apple Mango Orange

In the above example, <td>Apple</td>, <td>Mango</td>and <td>Orange</td> are table cells.

Note: A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.

Table Heading <th>

The <th>tag is used to define a table header. It is generally the top row of the table. 

                  <table border="1">
                    <tr>
                      <th>Item</th>
                      <th>Count</th>
                    </tr>
                    <tr>
                      <td>Mango</td>
                      <td>125</td>
                    </tr>
                    <tr>
                      <td>Orange</td>
                      <td>75</td>
                    </tr>
                  </table>

Output:

Item Count
Mango 125
Orange 75

In the above example, Item and Count are table headers and they are used to represent the category of data in a particular row.

Here, the styling of the table headers is bold and center-aligned. This is because the tag has some default styling.

1 - Table Parts

The HTML table can be divided into three parts: a header, a body, and a footer.


  • Table Header (<thead>)

    • We use the <thead> tag to add a table head. The <thead> tag must come before any other tags inside a table. 

                        <table>
                    <thead>   
                        <tr>
                          <th>Head1</th>
                          <th>Head2</th>
                        </tr>
                    </thead>   
                    </table>

    The content of <thead> is placed on the top part of the table and we usually place the rows with table headers inside the <thead> tag.


  • Table Body (<tbody>)

    • We use the <tbody> tag to add a table body. The <tbody> tag must come after <thead> and before any other tags inside a table. 

                        <table>
                      <thead>
                        ...
                      </thead>
                      <tbody>   
                        <tr>
                          <td>Cell 1</td>
                          <td>Cell 2</td>
                        </tr>
                      </tbody>   
                      ...
                    </table>
    The content of <tbody> is placed on the center part of the table and we usually place the rows with the content we want to represent in the <tbody>.

  • Table Footer (<foot>)

    • We use the <foot> tag to add a table footer.

    The <foot> tag must come after <foot> and before any other tags inside a table. 

                        <table>
                      <thead>
                        ...
                      </thead>
                      <tbody>
                        ...
                      </tbody>
                      <tfoot>   
                        <tr>
                          <td>foot 1</td>
                          <td>foot 2</td>
                        </tr>
                      </tfoot>   
                    </table>

    The content of <tbody> is placed on the bottom part of the table and we usually place the rows with the footer in the <foot>.

    All these tags must be placed inside a <table> tag and must contain at least one <tr>.

    Example: HTML Table Head, Body, and Footer 
                        <table> border="1"
                      <thead>
                        <tr>
                          <th>S.N</th>
                          <th>Item</th>
                          <th>Quantity</th>
                        </tr>
                      </thead>
                      <tbody>
                        <tr>
                          <td>1</td>
                          <td>Apple</td>
                          <td>2</td>
                        </tr>
                        <tr>
                          <td>2</td>
                          <td>Mango</td>
                          <td>2</td>
                        </tr>
                        <tr>
                          <td>3</td>
                          <td>Orange</td>
                          <td>1</td>
                        </tr>
                      </tbody>
                      <tfoot>
                        <tr>
                          <td></td>
                          <td>Total</td>
                          <td>5</td>
                        </tr>
                      </tfoot>
                    </table>

    Output:

     S.N   Item  Quantity 
     1  Apple  2
     2  Mango  2
     3  Orange  1
     Total  5
    2 - Colspan

    The colspan attribute merges cells across multiple columns.

                        <table>
                      <tr>
                        <th>S.N</th>
                        <th>Item</th>
                        <th>Quantity</th>
                      </tr>
                      <tr>
                        <td>1</td>
                        <td>Apple</td>
                        <td>2</td>
                      </tr>
                      <tr>
                        <td>2</td>
                        <td>Mango</td>
                        <td>2</td>
                      </tr>
                      <tr>
                        <td colspan="2">Total</td>
                        <td>4</td>
                      </tr>
                    </table>

    Output:

    S.N Item Quantity
    1 Apple 2
    2 Mango 2
    Total 4

    In the above example, you can see that the last row only has 2 cells with one cell occupying 2 columns.

    The value of the colspan attribute determines how many columns the cell occupies.

    3 - Rowspan

    The rowspan attribute merges cells across multiple rows.

                      <table border = "1">
                    <tr>
                      <th>Name</th>
                      <th>Subject</th>
                      <th>Marks</th>
                    </tr>
                    <tr>
                   <td rowspan="3">Mark Smith</td>   
                      <td>English</td>
                      <td>67</td>
                    </tr>
                    <tr>
                      <td>Maths</td>
                      <td>82</td>
                    </tr>
                    <tr>
                      <td>Science</td>
                      <td>91</td>
                    </tr>
                  </table>

    Output:

    Name Subject Marks
    Mark Smith English 67
    Maths 82
    Science 91

    In the above example, you can see that the first column only has 2 cells with one cell occupying 2 rows.

    The value of the rowspan attribute determines how many rows the cell occupies.

    4 - Caption

    The <caption> element acts as the title of the table. It is used to give a short description of the table. It shows up on top of the table

    The <caption> tag must be the first child of the <table> element. 

                      <table border = "1">
                  <caption>Employee Details</caption>       
                    <tr>
                      <th>Name</th>
                      <td>Sam</td>
                      <td>Steve</td>
                      <td>Peggy</td>
                    </tr>
                    <tr>
                      <th>Age</th>
                      <td>31</td>
                      <td>42</td>
                      <td>29</td>
                    </tr>
                    <tr>
                      <th>Gender</th>
                      <td>M</td>
                      <td>M</td>
                      <td>F</td>
                    </tr>
                  </table>

    Output:

    Employee Details
    Name Sam Steve Peggy
    Age 31 42 29
    Gender M M F
    5 - Border

    Remember we have used the border attribute in our first example.

    In HTML, the border attribute is used to add a border to a table and all the cells.

    We can shape the table using css by giving the table a class named exampleTable .

    You can try the following codes by opening the style tag inside the head part of your HTML page.

    Below is a basic example of how to apply border styling to an HTML table using CSS: 

                      <!DOCTYPE html>
                  <html>
                  <head>
                  <style> 
                    .exampleTable {
                      border-collapse: collapse;
                      width: 100%;
                    }
                  
                    th, td {
                      border: 3px solid black;
                      padding: 8px;
                      text-align: left;
                    }
                  </style>    
                  </head>
                  <body>
                  
                  <table>
                    <tr>
                      <th>Product Name</th>
                      <th>Price</th>
                    </tr>
                    <tr>
                      <td>Pencil</td>
                      <td>$2.50</td>
                    </tr>
                    <tr>
                      <td>Notebook</td>
                      <td>$5.00</td>
                    </tr>
                  </table>
                  
                  </body>
                  </html>

    The table should look like this:

    Output:

    Product Name Price
    Pencil $2.50
    Notebook $5.00

    In this example, the border-collapse property is used to combine table borders, and the border property adds a 3-pixel thick black border to the cells.

    By using the padding property, a 8-pixel space is given between the data in the cell and the table.



    Now let's change the border properties and examine:

                        .exampleTable {
                      border-collapse: collapse;
                      width: 100%;
                    }
                    
                    th, td {
                      border: 3px dotted black;
                      padding: 8px;
                    }

    Output:

    Product Name Price
    Pencil $2.50
    Notebook $5.00

    These basic examples of HTML table border styling will give you an idea of how to customize the look of your tables using CSS.

    By using the extensive options of CSS, you can customize your table with many features such as adding background color, rounding corners.

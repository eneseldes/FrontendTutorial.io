IDs And Classes
What are These?

Selectors are used to identify elements and organize the web page. ID and class are a way to do this.

The way how to add a class or ID in HTML to an element is as follows: 

                    <div class="myClass">
                    <div class="myId">

After class or id value is set, these items can be selected from CSS. 

                    .myClass{
                        width: 100px;
                        height: 100px;
                        background-color: #7364d273;
                    }
                    #myId{
                        width: 50%;
                        height: 200px;
                        background-color: #858ae3b6;
                    }

With this way, we identified two divs in HTML.After that in CSS, we added features to these divs via the ID and class selectors.

Differences Between ID And Class

Although both of them are used to do same task, there is differences between them:

Class

  • Can be used to mark multiple elements in a page

  • Specificity level is 10.

  • May select a group of element.

ID

  • Can only be used to mark a single element in a page

  • Specificity level is 100.

  • Can only select one specific element.

Important Footnote

ID has also another functions. One of them is creating a correlation between the label and the form elements: 

                    <form action="">
                        <label for="correlation">Your Name Here:</label>
                        <input id="correlation" type="text">
                    </form>

If you click on 'Your Name Here' text, the textbox right of it will be focused. This is because label and input element is matched with ID.

ID has several more functions like above. Therefore, class is more proper to use as selector.

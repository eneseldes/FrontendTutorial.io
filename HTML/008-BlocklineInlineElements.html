Blockline And Inline Level Elements

Every element on HTML takes up space in the web page. There are two types of element depend on how they take place on web page:

1-Blockline Elements
2-Inline Elements

Blockline Elements

These elements occupy the entire width of its row although they are not completely fill it. An extra element cannot be contained next to blockline elements as you can see below:

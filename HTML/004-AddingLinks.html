Adding Links

Links are found in nearly all web pages. Links allow users to click their way from page to page.


Link with URL:

To add a link to another website, use the <a>(anchor) element with the href attribute, which indicates the link's destination.

Example:
 
                  <p><a href="https://www.aybubiltek.com" title="Goes to Biltek Website" >Visit Biltek Website</a></p> 

                  <p><a href="https://www.w3schools.com/html/">Website</a> that can help you in your HTML learning process. </p>

The code output will be like this:

  • Visit Biltek Website
  • Website that can help you in your HTML learning process.
    • Link with Target Attribute:

    By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

    The target attribute can have one of the following values:

    _self - Default. Opens the document in the same window/tab as it was clicked

    _blank - Opens the document in a new window or tab

    _parent - Opens the document in the parent frame

    _top - Opens the document in the full body of the window

    Example:

    To open a link in a new browser tab or window, use the target attribute with the value _blank. 

                        <p><a href="https://www.aybubiltek.com" target="_blank"> Visit Biltek Website (Opens in New Tab) </a></p>

    The code output will be like this:

  • Visit Biltek Website (Opens in New Tab)
    • Link to Download Items

    Add the item and the "download" attribute to download items via link.

    Example:
                      <p> Download <a href="images/biltek.png" download>Biltek Logo</a></p>

    The code output will be like this:

  • Download Biltek Logo
    • Link to an Email Address

    Use mailto: inside the href attribute to create a link that opens the user's email program (to let them send a new email):

    Example:
                     <p><a href="mailto:contact@example.com">Email Us</a></p>

    The code output will be like this:

  • Email Us (Clicking it opens the mail application. )
    • Link with Anchor (Jump Link):

    To create a link that jumps to a specific section on the same page, use the id attribute on the target element, and reference it with the # symbol in the href attribute.

    Example:
                      <div id="targetAttribute" >Link with Target Attribute: </div>
                              ...
                              ...
                  <p>Example: <a href="#targetAttribute">Jump to Target Attribute Part</a></p>

    The code output will be like this:

  • Jump to Target Attribute Part
    • Adding a Favicon

    A favicon is a small image displayed next to the page title in the browser tab. It is written in <head> section.

    You can use any image you like as your favicon. You can also create your own favicon on sites like https://www.favicon.cc.

    Example:
                     <link rel="icon" type="image/png" href="images/html_icon.png"></link>

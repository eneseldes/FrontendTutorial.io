HTML Introduction
What is HTML?

HTML stands for HyperText Markup Language.

It is the backbone of every web page, providing the structure and content for displaying information on the internet.

HTML uses a series of elements and tags to define the different components of a webpage, such as headings, paragraphs, images, links, tables, forms, and more.

A Simple HTML Document
  • All HTML documents must start with a document type declaration: <!DOCTYPE html>.
  • The HTML document itself begins with <html> and ends with </html>.
  • The visible part of the HTML document is between <body> and </body>. 
    •                   <!DOCTYPE html>
                  <html lang="en">
                    <head>
                      <meta charset="UTF-8" />
                      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                      <title>Document</title>
                    </head>
                    <body>
                      <h1>My First Heading</h1>
                      <p>My first paragraph.</p>
                    </body>
                  </html>
    HTML ELEMENTS

    An HTML page is composed of various elements and tags that define its structure, content, and presentation. The basic structure of an HTML page consists of two main sections: the <head> and the <body>.

    1 - Head Section


    The <head> section is not visible on the web page itself but contains important metadata and information about the page. It usually includes:

    <!DOCTYPE html> declaration represents the document type, and helps browsers to display web pages correctly.

    Title (<title>): The title of the webpage.

    Link Tags (<link>): These tags are used to link external resources to the HTML page, such as stylesheets (CSS files), favicons, and fonts.

    Meta Tags (<meta>): These tags provide metadata about the page, such as the character encoding, author information, viewport settings for responsive design, and more. For example:

  • The charset meta tag specifies the character encoding like "UTF-8".
  • The expression "width=device-width" specifies that the page will be the same width as the device.
  • The statement "initial-scale=1.0" ensures that the page is displayed at its original size.

    • 2 - Body Section


    The <body> section contains the visible content of the webpage that users see in their browsers. This includes text, images, videos, links, forms, and any other elements that make up the actual content of the webpage.
